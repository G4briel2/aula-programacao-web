# Definição de escopo, prototipagem e UX

## Diferentes tipos de websites

- Institucional -> Um site para mostrar seus serviços
- Blog -> Site para divulgação de notícias / artigos
- E-commerce -> Site de vendas
- Landing Page -> Site para vender seu produto
- Plataforma -> Tudo que não são os outros

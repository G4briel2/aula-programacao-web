# Atividade – Hospedagem e CMS

# Ferramenta Utilizada

A plataforma escolhida foi o **Pantheon**, que oferece hospedagem gratuita para projetos web e suporte a CMS como o WordPress.

---

# Desenvolvimento da Atividade

## Instalação do CMS

Foi utilizado o **WordPress** como CMS.

- Instalação realizada pela própria plataforma  
- Configuração inicial concluída  

---

## Desenvolvimento do Site

O projeto desenvolvido foi um **site de venda de motos**, contendo:

- Página inicial personalizada  
- Estrutura básica de navegação  
- Inserção de conteúdos e textos  
- Ajustes visuais com temas  

---

## Personalização

Foram realizadas algumas personalizações, como:

- Instalação de tema para definir o layout  
- Alteração de textos e elementos visuais  
- Organização da página inicial  

---

# Dificuldades

- Não é possível ter a liberdade total de personalização 
- Dependência de temas e plugins para alterações mais avançadas  
- Interface limitada para personalização detalhada  

---

# Prós x Contras

## Pontos positivos
- Facilidade de uso  
- Rapidez na criação de sites  
- Não exige conhecimento avançado em programação

## Pontos negativos
- Baixa flexibilidade em alguns casos  
- Dependência de ferramentas externas (plugins)  
- Menor controle técnico sobre o sistema
